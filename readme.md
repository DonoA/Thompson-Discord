Thompson-Discord
=====

A discord bot servant.

Features:
- Basic permission system
  - Promotion
  - Demotion
- Stream/channel stats fetcher
  - Live view
- User stalker
  - Attempt to find other accounts tied to this user on the web
- Code executor
- Markov simulator
  - Users
  - Reddit
  - YouTube
  - Twitch
- Self update task
- Watch field on URL
